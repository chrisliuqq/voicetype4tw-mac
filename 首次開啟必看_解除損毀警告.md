# 安裝與解除「已損毀」警告教學

由於本應用程式尚未向 Apple 支付開發者憑證年費 (Apple Developer Program)，因此當您從網路下載並解壓縮本應用程式後，macOS 的 Gatekeeper 安全機制會自動將其標記為「已損毀」或「無法打開」。

這是所有開源或免費 Mac 軟體的常見情況，請放心，這**並非病毒**。您只需要透過一行簡單的終端機指令，解除 macOS 的隔離保護 (Quarantine) 即可永久正常使用。

### 🚀 簡單 3 步驟搞定：

1. **移動到應用程式資料夾**
   請將解壓縮出來的 `VoiceType4TW-Mac.app` 拖曳到您的「應用程式 (Applications)」資料夾。
   *(這一步很重要，確保之後捷徑不會跑掉)*

2. **打開終端機 (Terminal)**
   您可以按下 `Command + 空白鍵` 叫出 Spotlight，輸入 `Terminal` 或 `終端機` 並按下 Return (Enter) 打開它。

3. **複製貼上解除指令**
   在終端機中，完整複製並貼上以下指令，然後按下 Return (Enter)：

   ```bash
   xattr -cr /Applications/VoiceType4TW-Mac.app
   ```

   *(如果您將 App 放在其他資料夾，請將指令後面的路徑替換成您實際放置的位置，或者先輸入 `xattr -cr ` 然後把 App 拖曳進終端機視窗，會自動輸入正確路徑)*

---
**🎉大功告成！**
現在您可以從 Finder 或 Launchpad 中正常開啟 `VoiceType4TW-Mac.app`，享受光速般的語音辨識體驗了！
不會再跳出煩人的警告視窗。
